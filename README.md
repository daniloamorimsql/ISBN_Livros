# ISBN_Livros
Mais de 200.00 livros registrados no Brasil, disponíveis em apenas um banco de dados para usar em seus labs e estudos.

## Aviso sobre os dados

Os dados desta base foram obtidos a partir de fontes públicas,
principalmente através da BrasilAPI.

Este projeto não reivindica propriedade sobre as informações
bibliográficas ou conteúdos associados.

Caso algum detentor de direitos identifique material que não
deveria estar distribuído, abra uma issue ou solicite remoção.
